get /orders
Returns all orders depending on params (ordersGet)
If no param if specified, orders from current day will be fetched. Before and after filters together and &quot;at&quot; add the date to the filter result

Query parameters

at (optional)
Query Parameter — Filter orders based on exact ISO date. format: date
before (optional)
Query Parameter — Filter orders before or equal ISO date. format: date
after (optional)
Query Parameter — Filter orders after or equal ISO date. format: date

Return type

array[order]
